## ⌨️ Development

## 🛠 Technology Stack

* Java 8 *JDK 1.8*
* Maven
* Git
* IDE (Netbeans)

### Clone Code

```bash
git clone git@github.com:JhonatanMedeiros/students-project.git
```

### Install Dependencies

```bash
mvn clean install
```

### Compile

```bash
mvn compile
```
